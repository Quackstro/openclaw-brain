### Optimized Budget Path (~$650-750)
- Budget Pixhawk clone ($35 vs $100)
- Use Pixel 9 Pro as ground robot brain initially (save $80)
- Use spare laptop as hub (save $55-83)
- Bundle soldering kit deals (save $30-40)

### Notes
- LiPo charger (iMAX B6AC or ToolkitRC M6) handles BOTH ground robot 3S and drone 4S batteries
- XT60 connectors shared across both projects
- Tools are a one-time investment — reused for all future builds
- 3D printing cost is filament only thanks to neighbor's printer
- PETG recommended over PLA for outdoor durability (UV + temp resistance)
