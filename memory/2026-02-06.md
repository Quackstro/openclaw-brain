### Memory Function Design
Standard calculator memory with time values:
- **M+** (Memory Add): Add current display value to memory
- **M-** (Memory Subtract): Subtract current display from memory  
- **MR** (Memory Recall): Display stored memory value
- **MC** (Memory Clear): Reset memory to 0:00

Use case: User tracking multiple time totals, can store subtotal then continue calculating.
