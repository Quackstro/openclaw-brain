## Brain CLI Issues
- The brain CLI (`cli-impl.ts`) has its own embedder with `dim=1536` and OpenAI-compatible base URL — doesn't match the plugin's Gemini embedder (`dim=3072`)
- CLI `classify` and `delete` commands fail because of this mismatch
- Workaround: Use direct TypeScript scripts importing `store.js` with correct dim (3072)
- LanceDB deletes are lazy — vector search returns deleted records until compaction; `store.list()` correctly filters them
