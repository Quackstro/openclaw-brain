
## Anthropic Usage Monitor Setup
- Created `/home/clawdbot/clawd/scripts/anthropic-usage-monitor.mjs`
- Monitors 5-hour and 7-day usage windows via Anthropic OAuth usage API
- Sends Telegram notifications when crossing: 25%, 50%, 75%, 90%, 95%, 99% thresholds
- Also notifies when limits reset (usage drops significantly)
- Handles OAuth token refresh automatically (updates auth-profiles.json)
- State stored in `.usage-monitor-state.json`
- Cron: runs every 5 minutes via system crontab
- Log: `/home/clawdbot/clawd/scripts/usage-monitor.log`
- Current usage: 5-Hour = 25%, 7-Day = 50%
